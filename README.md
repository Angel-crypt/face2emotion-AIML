# Face2Emotion (Clasificación de Emociones Faciales)

Proyecto enfocado en la clasificación de expresiones faciales en tres emociones básicas (**Feliz, Triste, Enojado**).

Este repositorio documenta el desarrollo del proyecto estructurado para garantizar la calidad del procesamiento y la solidez del entrenamiento.

---

## Estado del Proyecto por Iteraciones

Delimitación del problema, Análisis Exploratorio de Datos (EDA) y preprocesamiento de imágenes.

---

## Inicio Rápido (Quick Path)

Para ejecutar este proyecto en tu entorno local o en Google Colab, seguí estos pasos:

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/Angel-crypt/face2emotion-AIML.git
   ```
2. **Descargar el Dataset**:
   * Descargá el dataset `samithsachidanandan/human-face-emotions` desde Kaggle.
   * Alternativamente, el notebook se encarga de descargarlo automáticamente mediante `kagglehub`.
3. **Ejecutar el Notebook**:
   * Abre `face2emotion_AIML.ipynb`.
   * Ejecutá las celdas secuencialmente para descargar el dataset, balancear las clases, preprocesar las imágenes y dividir los conjuntos de datos en memoria.
